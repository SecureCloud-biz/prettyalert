# Prettyalert
A pretty modal window.

## Usage
Succes message :
```javascript
sprettytalert('Hi there', 'And this is the message','succes');
```
Error message :
```javascript
sprettytalert('Hi there', 'And this is the message','error');
```
Warning message :
```javascript
sprettytalert('Hi there', 'And this is the message','attention');
```
