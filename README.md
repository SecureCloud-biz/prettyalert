# Prettyalert
A pretty modal window.

## Usage
Succes message :
```javascript
prettytalert('Hi there', 'And this is the message','succes');
```
Error message :
```javascript
prettytalert('Hi there', 'And this is the message','error');
```
Warning message :
```javascript
prettytalert('Hi there', 'And this is the message','warning');
```
